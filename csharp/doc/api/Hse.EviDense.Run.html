Executes the next step in the run's state machine:

  • BASELINE: records a new baseline and transitions to AIR.
  • AIR: measures the air sample and transitions to SAMPLE.
  • SAMPLE: measures the sample, stores it with metadata, and returns to BASELINE.
Saves the updated JSON after each step.
